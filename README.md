# AI与安全运营：
一本将传统安全运营与AI前沿技术深度融合的实战指南

## 🔍 项目简介
旨在帮助安全从业者理解并掌握AI驱动的安全运营技术。通过理论与实践相结合的方式，系统地介绍了从传统安全运营到AI原生安全运营的演进路径。

## 🎯 核心特色
### 理论深度
- 深入剖析AI在安全运营中的应用场景
- 系统讲解AI原生安全运营的技术架构
- 全面覆盖AI安全治理与风险管理
### 实战导向
- 代码级实操示例 ：Sigma规则、SPL/KQL查询、SOAR剧本
- 真实案例分析 ：Log4j漏洞响应、银行告警分诊、勒索软件处置
- 工具链配置指南 ：Trivy、Syft、StackStorm、OPA等
### 体系完整
- 从资产管理到威胁检测的全流程覆盖
- 从初级工程师到架构师的多层次内容
- 从技术实现到合规治理的全方位视角

## 📊 核心内容亮点
### 1. AI原生安全运营架构
- 智能体架构设计（感知→推理→决策→行动）
- RAG流水线实现与向量数据库选型
- 多智能体协同协议与通信机制
### 2. 实战代码示例
- Sigma规则 ：横向移动检测、暴力破解检测
- SPL/KQL查询 ：多源关联分析、攻击链追踪
- SOAR剧本 ：SSH暴力破解响应、勒索软件处置
- Python脚本 ：资产查询、防火墙封禁、工单创建
### 3. 行业案例研究
- 金融行业 ：反欺诈AI应用、告警分诊实战
- 医疗行业 ：数据安全合规、PHI保护
- 制造业 ：ICS安全、工业协议防护
- 能源行业 ：OT/ICS安全整合
### 4. 合规与治理
- NIST AI风险管理框架应用
