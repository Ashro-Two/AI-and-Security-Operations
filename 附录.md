## 附录B AI安全运营关键术语表

### B.1 AI基础术语

| 术语 | 定义 |
|------|------|
| AI（人工智能） | 使计算机系统能够执行通常需要人类智能的任务 |
| ML（机器学习） | AI的子集，使系统能够从数据中学习 |
| DL（深度学习） | ML的子集，使用神经网络进行学习 |
| LLM（大语言模型） | 基于Transformer架构的大型语言模型 |
| 智能体（Agent） | 能够自主感知、推理、决策和行动的软件实体 |

### B.2 安全运营术语

| 术语 | 定义 |
|------|------|
| SOC（安全运营中心） | 集中处理安全监控和响应的中心 |
| SIEM（安全信息与事件管理） | 收集和分析安全日志的平台 |
| SOAR（安全编排自动化与响应） | 自动化安全响应的平台 |
| EDR（终端检测与响应） | 终端层面的威胁检测和响应 |
| NDR（网络检测与响应） | 网络层面的威胁检测和响应 |

### B.3 AI安全术语

| 术语 | 定义 |
|------|------|
| 提示注入（Prompt Injection） | 通过恶意输入误导AI模型 |
| 模型漂移（Model Drift） | 模型性能随时间下降 |
| 对抗样本（Adversarial Example） | 特意设计的输入以误导模型 |
| 数据泄露（Data Leakage） | 训练数据或敏感信息的泄露 |
| AI智能体失控（Rogue Agent） | AI智能体执行未授权操作 |

## 附录E 安全运营成熟度自评表

### E.1 成熟度评估维度

| 维度 | Level 1 | Level 2 | Level 3 | Level 4 | Level 5 |
|------|---------|---------|---------|---------|---------|
| **资产管理** | 无清单 | 基本清单 | 分类分级 | 全生命周期 | AI驱动 |
| **漏洞管理** | 被动响应 | 定期扫描 | 风险评估 | 自动化流程 | AI预测 |
| **威胁检测** | 人工监控 | 规则告警 | 威胁狩猎 | AI检测 | 智能预测 |
| **应急响应** | 无预案 | 基本预案 | 定期演练 | 自动化响应 | AI辅助 |
| **合规管理** | 被动合规 | 基本合规 | 合规计划 | 持续合规 | AI合规 |

### E.2 评分方法

1. 每个维度按成熟度等级评分（1-5分）
2. 计算平均得分
3. 根据平均得分确定整体成熟度等级
